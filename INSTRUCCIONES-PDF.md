# Cómo Generar el PDF del Manual

Se ha creado un manual HTML completo en: `manual-venta-garaje.html`

## Método 1: Desde el Navegador (Recomendado)

1. **Abre el archivo HTML:**
   - Doble clic en `manual-venta-garaje.html`
   - O arrastra el archivo a tu navegador (Chrome, Firefox, Edge, Safari)

2. **Imprimir a PDF:**
   - **Windows/Linux:** Presiona `Ctrl + P`
   - **Mac:** Presiona `Cmd + P`

3. **Configuración de impresión:**
   - Destino: Selecciona **"Guardar como PDF"** o **"Microsoft Print to PDF"**
   - Diseño: **Vertical**
   - Márgenes: **Predeterminados** o **Ninguno**
   - Escala: **Predeterminado** (100%)

4. **Guardar:**
   - Haz clic en **"Guardar"** o **"Imprimir"**
   - Nombra el archivo: `Manual-Venta-Garaje.pdf`
   - ¡Listo! Tu PDF de 5 páginas está generado

## Método 2: Desde la Terminal (Si tienes wkhtmltopdf instalado)

```bash
wkhtmltopdf manual-venta-garaje.html Manual-Venta-Garaje.pdf
```

## Método 3: Online

1. Visita: https://www.html2pdf.app/
2. Sube el archivo `manual-venta-garaje.html`
3. Haz clic en "Convert to PDF"
4. Descarga el PDF generado
